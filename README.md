# Макроэкономический анализ данных
Скрипт для обработки CSV-файлов с экономическими данными. Формирует отчёт со средним ВВП по странам.

## Использование
python main.py --files data/sample_data.csv --report average-gdp

Параметры:
--files — один или несколько CSV-файлов для анализа
--report — тип отчёта (average-gdp)
