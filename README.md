# Макроэкономический анализ данных
Скрипт для обработки CSV-файлов с экономическими данными. Формирует отчёт со средним ВВП по странам.

## Использование
python main.py --files data/sample_data.csv --report average-gdp

Параметры:
--files — один или несколько CSV-файлов для анализа
--report — тип отчёта (average-gdp)

## Формат данных
CSV-файл должен содержать колонки country и gdp.

## Тестирование
Проект покрыт тестами pytest (12 тестов).

Проверяют:
- обработку пустых данных
- расчет среднего для одной страны
- расчет для нескольких лет
- сортировку нескольких стран
- обработку некорректных значений
- пропущенные поля
- работу фабричной функции
- обработку аргументов командной строки

## Структура проекта
main.py — точка входа, обработка аргументов
analysis.py — анализаторы данных (базовый класс и реализация)
read.py — чтение CSV-файлов
tests/ — директория с тестами
data/ — примеры данных
